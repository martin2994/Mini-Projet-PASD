void interprete_chunk ( chunk  ch,
interpretation_context  ic 
)

Interpret a single chunk in a context.

  • if it is a value, it should be stacked;
  • if it is an operator, it should be evaluated in this context and destroyed after.

If a basic_type_error is returned by the evaluation of an operator, then the first value on the stack should be a value_error. This value is printed (and destroyed) and then the stack id printed on stderr as follows: 

### ERROR ### operator pop ###  --error-- # 6
vvvvvvvv stack  top  vvvvvvvvvv
^^^^^^^^ stack bottom ^^^^^^^^^
Parameters
chchunk to interpret
iccontest to interpret it
