Detailed Description

Operator label: when evaluated it is replaced by the corresponding value from the dictionary.

If the associated value is a value_block, then the chunk's are processed in order. Otherwise, it is processed as usually.

Label can be created even if the entry is not defined in the dictionary. It should be defined when evaluated.

assert is enforced.
