Detailed Description

Provide all the necessary to handle value_error.

A value_error indicates an error with a code corresponding to its nature. The function error_get_value returns this value (as a basic_type)

This value cannot appear in any input. A value_error can only appear an incorrect program reading, interpretation or evaluation.

Output is like –error– # followed by the error number.

assert is enforced.
