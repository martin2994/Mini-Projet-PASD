Detailed Description

Provide a 'universal' elementary type able to store any 'basic' value.

Use

  • long_long_int: any integer type (including unsigned, char, size...)
  • long double: any float type
  • pointer: any pointer
  • bool and constant basic_type_true and basic_type_false: booleans
  • void constant: for void (i.e. nothing)
  • error constant: for error

It should be used as a basic value and passed as is, avoiding any pointer/reference. (this is why there is no basic_type_destroy available).

Cela ressemble au cours, même s'il y a des différences.

assert is enforced to test that the get_xxx are used on the appropriate type.
