Detailed Description

Comprehensive listing of all labels that are reserved word together with functions to create the corresponding operator's.

Only operator's written as a sequence of letters are concerned (i.e. neither + nor &&… are concerned).

This is used to:

  • distinguish keyword from legal label, and
  • to identify and generate the right operator on reading.

This module has only data and no function.
