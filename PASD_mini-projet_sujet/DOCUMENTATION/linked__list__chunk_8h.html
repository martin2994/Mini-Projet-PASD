Detailed Description

Structure used to store list of chunk 's.

Please note that linked_list_chunk does not make any copy of chunk's.

The list is bidirectional and can be accessed from both end.

The structure is 2-level:

  • link's are used to provide the bidirectional links (with pointers) and points to chunk
  • the main structure has pointers to first and last link's.

The empty list has the pointers of its main structure at NULL.

bool linked_list_chunk_add_self_copy_front ( linked_list_chunk  llc,
unsigned int  k 
)

Add a copy of the k first chunk at the beginning of the linked_list_chunk to it-self. If there is less than k chunk then no copy is made.

For k, the following linked_list_chunk 

[front]                                     ch0 ch_1 ch_2 ...  ch_k-2 ch_k-1   ch_k ch_k+1 ch_k+2

is transformed into 

[front]  ch0 ch_1 ch_2 ...  ch_k-2 ch_k-1   ch0 ch_1 ch_2 ...  ch_k-2 ch_k-1   ch_k ch_k+1 ch_k+2
Parameters
llclinked_list_chunk to add to
ksize of the prefix to copy in front
Precondition
llc is valid (assert-ed)
Returns
false if there where less than k element. In such a case, no copy is made.
