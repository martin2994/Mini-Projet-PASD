Detailed Description

Provide a dictionary (associative table).

  • keys must be non-empty sstring's
  • values must be chunk's

Keys and values are copied to get independent long term storage. In the same spirit, queried values are copies. The caller is in charge of destroying copies; this is indicated by the _copy in the query function name.

Values are stored in an ordered binary tree. They are two data structures: one for the dictionary and one for nodes.
