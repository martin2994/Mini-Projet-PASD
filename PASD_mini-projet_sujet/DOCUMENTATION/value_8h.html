Detailed Description

Chunk used to hold any value.

Except for value_block, all value's are immutable (i.e. cannot be modified).

A chunk is a value if it gives a positive response to the message MESSAGE_VALUE_IS_VALUE.

value's must also respond to the message MESSAGE_VALUE_GET_VALUE.

There is no …_create This it thought as an abstract intermediary class.

assert is enforced.
