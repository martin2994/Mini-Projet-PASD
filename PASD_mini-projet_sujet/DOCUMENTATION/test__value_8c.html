Detailed Description

Basic tests forvalue's.

Test copy, print and destroy on:

  • value_block
  • value_boolean
  • value_double
  • value_error
  • value_int
  • value_protected_label
  • value_sstring

value_block is not tested since it relies on linked_list_chunk. It is tested in test_linked_list_chunk.c.

All value's (including value_block) are also tested with pf programs.

This should also be used to test for memory leak.

assert is enforced.
