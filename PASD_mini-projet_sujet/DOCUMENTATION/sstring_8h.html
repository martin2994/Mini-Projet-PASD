Detailed Description

This module provides a « safer » string.

These are not C-string but a struct that holds the length of the string and a pointer to the actual char sequence. Please note that there is no '\0' to mark the end of the sstring.

Empty string is encoded by 0 length and NULL pointer.

assert is enforced.
