Detailed Description

chunk used to hold any operator.

operator's are immutable (i.e. cannot be modified).

A chunk is a operator if it gives a positive response to the message MESSAGE_OPERATOR_IS_OPERATOR.

Any operator must also respond to the message MESSAGE_OPERATOR_EVALUATE.

There is no operator_create. This it thought as an abstract intermediary class.

assert is enforced.
